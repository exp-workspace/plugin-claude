# plugin-claude

A Claude Code plugin with workspace and migration skills.

## Install

```
/plugin install https://github.com/exp-workspace/plugin-claude
```

## Skills

| Command | Description |
|---|---|
| `/plugin-claude:setup-workspace` | Set up a new development workspace |
| `/plugin-claude:migrate-code` | Migrate code between versions, frameworks, or patterns |
| `/plugin-claude:update-config` | Update configuration files and settings |
