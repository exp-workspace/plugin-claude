# experience-workspace

A Claude Code plugin with workspace and migration skills.

## Install

Run these two commands in Claude Code in order:

**1. Add the marketplace:**
```
/plugin marketplace add exp-workspace/plugin-claude
```

**2. Install the plugin:**
```
/plugin install experience-workspace
```

## Skills

| Command | Description |
|---|---|
| `/experience-workspace:setup-workspace` | Set up experience workspace for your project |
| `/experience-workspace:migrate-code` | Migrate code to add quick-edit support |
| `/experience-workspace:update-config` | Update your project config to open experience workspace automatically |
