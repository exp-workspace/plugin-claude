# experience-workspace

A Claude Code plugin with workspace and migration skills.

## Install

```
/plugin marketplace add exp-workspace/plugin-claude
/plugin install experience-workspace@exp-workspace/plugin-claude
```

## Skills

| Command | Description |
|---|---|
| `/experience-workspace:setup-workspace` | Set up a new development workspace |
| `/experience-workspace:migrate-code` | Migrate code between versions, frameworks, or patterns |
| `/experience-workspace:update-config` | Update configuration files and settings |
